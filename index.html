Frequently Asked Questions

How do I take a screenshot for translation?

+

You can take a screenshot in three ways:

  • Click the floating icon on your desktop
  • Use the status bar menu and select "Screenshot OCR"
  • Use the keyboard shortcut Shift+Cmd+S

After taking a screenshot, select the area containing text you want to translate.

Why is the app asking for screen recording permission?

+

AITrans needs screen recording permission to capture screenshots for text recognition. To grant permission:

  1. Go to System Preferences > Security & Privacy > Privacy
  2. Select "Screen Recording" from the left sidebar
  3. Check the box next to AITrans
  4. Restart the app if needed

What languages are supported for translation?

+

AITrans supports translation between multiple languages including:

  • Chinese (Simplified & Traditional)
  • English
  • Japanese
  • Korean
  • Thai
  • Vietnamese
  • Spanish
  • French
  • German

How do I configure my AI API key?

+

To set up your AI API key:

  1. Right-click the AITrans icon in your status bar
  2. Select "Settings" or "API Configuration"
  3. Enter your API key for the desired AI service
  4. Save the configuration

Supported AI services include Zhipu AI and Google Translate.

Can I use AITrans without an internet connection?

+

AITrans requires an internet connection for:

  • AI-powered analysis and detailed explanations
  • Translation services
  • OCR text recognition (uses local processing)

However, the app caches recent translations for offline viewing.

How do I change the floating icon position?

+

To reposition the floating icon:

  1. Right-click the AITrans icon in your status bar
  2. Go to "Floating Icon" settings
  3. Choose from: Bottom Right, Bottom Left, Top Right, or Top Left
  4. The icon will automatically move to the selected position

Troubleshooting

App won't start or crashes on launch

Solution:

  1. Make sure you're running macOS 10.15 or later
  2. Check if the app has necessary permissions in System Preferences
  3. Try restarting your Mac
  4. Reinstall the app if the problem persists

Screenshot function not working

Solution:

  1. Grant screen recording permission in System Preferences
  2. Make sure no other apps are using the screenshot function
  3. Try using the keyboard shortcut Shift+Cmd+S
  4. Check if the floating icon is visible and clickable

Translation results are inaccurate

Solution:

  1. Ensure your API key is valid and has sufficient credits
  2. Try selecting a smaller, clearer text area
  3. Check your internet connection
  4. Verify the source language is set correctly

Floating icon not visible

Solution:

  1. Check if the floating icon is enabled in settings
  2. Look for the icon in all four corners of your screen
  3. Restart the app
  4. Check if any other apps are covering the icon
